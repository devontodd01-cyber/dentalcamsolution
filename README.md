# Dental CAM Solution — Website

Canada's dedicated source for DS CAM software and professional-grade milling tools.

## Pages

- `index.html` — Landing page
- `software.html` — DS CAM Software features & pricing
- `videos.html` — Workflow video library
- `tooling.html` — Milling tools (links to AS Dental Tool)
- `about.html` — Why choose us
- `contact.html` — Contact form

## Notes

- All pages are fully self-contained — logo is embedded as base64, no external assets needed
- Video files stream directly from dental-softworks.com
- Milling tool cards link directly to product pages on asdentaltool.com

## Deploy

Connected to Vercel — any push to `main` branch auto-deploys to production.
